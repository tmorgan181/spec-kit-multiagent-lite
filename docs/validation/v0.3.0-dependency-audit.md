**Current behavior**:

**Problems**:

- âŒ Doesn't link to spec-kit docs/repo
- âŒ Doesn't mention Node.js requirement

**Fix for v0.3**:
```python

Learn more: https://github.com/github/spec-kit
"""
```
